# Sistema de Monitoramento - Super Centro Brasil

Sistema de gerenciamento e acompanhamento de instituições do Super Centro Brasil para Diagnóstico de Câncer.

## 🚀 Deploy no Vercel

1. Instale o Vercel CLI (opcional):
```bash
npm i -g vercel
```

2. Faça deploy:
```bash
vercel
```

Ou simplesmente:
- Acesse https://vercel.com
- Conecte seu repositório GitHub
- O deploy será automático!

## 💻 Desenvolvimento Local

```bash
npm install
npm run dev
```

Abra http://localhost:3000

## 📦 Build para Produção

```bash
npm run build
```

Os arquivos estarão em `/out`
